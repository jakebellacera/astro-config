# Basic Example

This is a basic example of an Astro site using the `astro-config` integration.

## Running the Example

```bash
# From the root of the monorepo
pnpm install
pnpm build

# From this directory
pnpm dev
```

## What's Included

- Astro site configured with the `astro-config` integration
- Custom configuration passed to the integration
- Example page demonstrating the integration usage
